>이것은 TanStack Router와 MDX를 함께 사용하는 테스트 포스트입니다.

## 소개

이 포스트는 다음과 같은 기능들을 테스트합니다:

- MDX 파일이 동적 라우트로 렌더링되는지
- 컴포넌트가 MDX 안에서 작동하는지
- 타입스크립트 지원이 잘 되는지

## 인터랙티브 컴포넌트

<CustomButton text="클릭해보세요!" />

## 코드 예제

```typescript
const greeting = (name: string) => {
  return `안녕하세요, ${name}님!`
}

console.log(greeting('개발자'))
```

## 마무리

TanStack Router와 MDX의 조합으로 강력한 블로그 시스템을 만들 수 있습니다.

export function CustomButton({ text }) {
  return (
    <button 
      className="bg-blue-500 hover:bg-blue-700 text-white font-bold py-2 px-4 rounded"
      onClick={() => alert('MDX에서 컴포넌트가 작동합니다!')}
    >
      {text}
    </button>
  )
}
